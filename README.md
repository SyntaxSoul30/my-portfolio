# Portfolio

Personal portfolio website built with PHP, HTML and CSS showcasing my skills, projects, experience and contact form.

## Files

- `index.php` — Main page and form handler
- `contact.php` — Contact form email processing
- `styles.css` — Site styling

## Setup

Deploy on a PHP-compatible web server. Update email in `contact.php` to receive contact form messages.
